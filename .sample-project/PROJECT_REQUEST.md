# Sample Project: Task Manager CLI

## Description
A simple command-line task management application that allows users to create, list, update, and delete tasks. The application should persist tasks to a local JSON file.

## Requirements
1. Create tasks with title, description, and priority
2. List all tasks with filtering options
3. Update task status (todo, in-progress, done)
4. Delete tasks
5. Data persistence using JSON file

## Target Platform
Python 3.10+ CLI application

## Testing Request
This project will be used to validate the multi-agent SDLC system.
